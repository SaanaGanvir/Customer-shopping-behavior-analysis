# 🛒 Customer Shopping Behavior Analysis

## 📊 Overview

This project analyzes customer shopping behavior using a dataset of 3,900 transaction records to identify purchasing patterns, spending trends, and customer preferences. The goal is to generate insights that can help businesses improve marketing strategies, customer engagement, and revenue decisions.

## 🗂️ Dataset

The dataset contains customer transaction information including **age, gender, location, product category, purchase amount, discount usage, subscription status, shipping type, and review ratings**. It was used to study purchasing behavior and customer segmentation.

## 🛠️ Tools & Technologies

* 🐍 **Python (Pandas)** – Data loading, cleaning, and preprocessing
* 🗄️ **PostgreSQL** – SQL queries for analytical insights
* 📈 **Power BI** – Interactive dashboard and visualizations
* 📑 **Gamma** – Report and presentation creation

## ⚙️ Project Workflow

1. **📥 Data Preparation** – Dataset imported and explored using pandas. Missing values were handled and columns were formatted.
2. **🔎 Exploratory Data Analysis (EDA)** – Identified trends in spending, product preferences, and customer behavior.
3. **💾 SQL Analysis** – Queries were used to analyze revenue patterns, subscription behavior, and high-value customers.
4. **📊 Dashboard Development** – Power BI was used to create an interactive dashboard to visualize insights.
5. **📝 Reporting** – Findings were summarized in a presentation and report.

## 🔑 Key Insights

* Spending behavior varies across **customer segments and demographics**.
* **Subscription users and repeat buyers** show different purchasing patterns.
* Certain **products and categories generate higher customer ratings and revenue**.
* Discount usage influences purchasing behavior in specific product groups.

## 📊 Dashboard

The **Power BI dashboard** provides visual insights into:

* 👥 Customer segments
* 💰 Revenue distribution
* 📦 Product performance
* 📉 Purchasing trends
